Arquitectura Principal del Sistema

💡 Leyenda del Diagrama

Flujo de Datos Principal

✅ Flujo Completo

El diagrama muestra los 3 flujos principales:

  1. Generación: Usuario → Dashboard → API → Test Generator → OpenAI → PostgreSQL
  2. Ejecución: Usuario → Dashboard → API → Test Executor → Playwright → Sample App
  3. Resultados: Playwright → Test Executor → PostgreSQL → Jira → Dashboard

Arquitectura de Despliegue (Docker)

💡 Características del Despliegue

  • 7 contenedores trabajando juntos en una red Docker
  • Volúmenes persistentes para datos (PostgreSQL, Redis, RabbitMQ)
  • Health checks automáticos para cada servicio
  • Restart policies para recuperación automática
  • Variables de entorno para configuración flexible

Arquitectura de Seguridad

🛡️ Defense in Depth - 4 Capas de Seguridad

  • Capa 1 (Perímetro): Firewall, HTTPS, Rate Limiting, WAF
  • Capa 2 (Autenticación): JWT, OAuth 2.0, API Keys, RBAC
  • Capa 3 (Datos): Encriptación AES-256, Enmascaramiento, PII, Vault
  • Capa 4 (Aplicación): Validación, Protección SQL/XSS, Auditoría
