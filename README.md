
# QA Mobile Testing AI

Sistema completo de automatización de pruebas para aplicaciones móviles con
Inteligencia Artificial.

[Ver Dashboard](https://a15galaxyp-ui.github.io/qa-mobile-testing-ai/)

## Características

- Generación automática de casos de prueba con IA (GPT-4)
- Ejecutor de pruebas en tiempo real
- Dashboard interactivo con métricas
- Integración con Jira
- Gestión segura de datos (PII, enmascaramiento)
- Despliegue con Docker

## Tecnologías

- **Frontend:** HTML5, Tailwind CSS, Chart.js
- **Backend:** Python 3.11, FastAPI
- **Base de Datos:** PostgreSQL, Redis
- **IA:** OpenAI GPT-4
- **Testing:** Playwright, Appium, Maestro

## Roadmap

- Fase 1: Dashboard y generación IA (Completado)
- Fase 2: Integración con backend real (En progreso)
- Fase 3: Ejecución real en dispositivos
- Fase 4: Integración completa con Jira
